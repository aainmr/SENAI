Formas de implementar o CSS em uma página HTML

As três formas de implementar o CSS à uma página HTML são:

  1. Inline
  2. Interna
  3. Externa

    4. 1. CSS Inline:

    Este método usa < style > como um atributo, dentro da própria tag à qual se deseja aplicá-lo.

    O método de inserção de CSS externo consiste em salvar uma página .css com os atributos desejados e depois linkar ao html abaixo da tag < head >. Como ainda não abordamos este método em aula, apresentaremos apenas sua metodologia teórica.

    Suas vantagens são que por o CSS estar em um documento separado, a estrutura HTML será mais limpa e mais leve. Além disso, a mesma página .css pode ser usada em várias páginas.

    Suas desvantagens são que pode aumentar o tempo de carregamento da página, causando erros de processamento.
